# VibeCoding Browser

> 本地开发项目管理器 + 浏览器

一个专为前端开发者设计的工具，让你不用再记 `localhost:3000`，所有本地项目一个地方管理。

## 核心功能

### 1. 项目管理
- 自动记录所有本地开发项目
- 区分本地服务和在线域名
- 卡片式预览，一眼看到所有项目

### 2. 端口自动扫描
- 自动检测本地运行的服务
- 支持常用端口快速扫描 (3000, 5173, 8080...)
- 支持全面扫描 (3000-10000)
- 实时显示服务运行状态

### 3. 多项目快速切换
- Tab 标签页管理
- Cmd+T 快速搜索
- 像 IDE 一样管理多个项目

### 4. 编辑器集成
- 一键在 VSCode/Cursor/Windsurf 中打开项目
- 一键在终端中打开项目目录
- 项目和编辑器无缝切换

### 5. 开发工具
- 响应式设备预览 (手机/平板/桌面)
- 内置 DevTools
- 截图功能
- 后退/前进/刷新

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + T` | 搜索项目 |
| `Cmd/Ctrl + W` | 关闭当前标签 |
| `Cmd/Ctrl + R` | 刷新页面 |
| `Cmd/Ctrl + L` | 焦点到地址栏 |
| `Cmd/Ctrl + [` | 后退 |
| `Cmd/Ctrl + ]` | 前进 |
| `Cmd/Ctrl + Tab` | 下一个标签 |
| `Cmd/Ctrl + Shift + Tab` | 上一个标签 |

## 安装

### 开发环境
```bash
npm install
npm run start
```

### 打包
```bash
npm run build
npm run package
```

## 使用场景

- 本地开发多个项目，需要频繁切换
- 需要同时预览手机端和电脑端效果
- 记不住各个项目的端口号
- 想要快速在编辑器和浏览器之间切换
